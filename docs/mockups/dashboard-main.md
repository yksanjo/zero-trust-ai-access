# Zero-Trust Access Dashboard

## Layout

- **Header**: "Zero-Trust AI Access" with status indicator
- **Metrics Row**: Total identities, active sessions, compliance rate
- **Agent List**: Table of all machine identities
- **Risk Panel**: High-risk agents highlighted

## Agent Table Columns

- Agent ID
- Identity Status (certificate icon)
- Trust Score (0-100 with color)
- Last Access
- Risk Level Badge
- Actions (rotate, revoke buttons)
